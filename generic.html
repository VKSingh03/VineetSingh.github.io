KUKA LBR iiwa R820 Robot Modeling

In this project, a model based on KUKA LBR iiwa R820 is created in SOLIDWORKS and simulated using ROS Noetic and Gazebo

SKILLS Used: SOLIDWORKS, Forward and Inverse Kinematics, system dynamics modeling

  • A robot based on specification sheet of KUKA LBR iiwa R820 is designed in Solidworks with a custom designed end effector.
  • The model is converted to URDF and controllers are added for all the joints and a new URDF is generated using XACRO.
  • Algorithms for Forward and Inverse Kinematics is written from scratch using Python.
  • Publishers and Subscribers are written to appropriately control the robot.
  • Final simulation done in Gazebo where the robot picks up objects from one platform and places on another.

December 2022

LQR and LQG controller design for two pendulum gantry crane

In this project, a detailed study of a given non-linear system is done and multiple control techniques are designed to control the system.

SKILLS Learnt: Kalman Filter and its implementation, non linear system modeling, linearization of non linear systems, Euler Lagrange system dynamics

  • This project is done in MATLAB, Simulink, and Python.
  • Euler Lagrange model for dynamic motion is derived for a double pendulum gantry crane.
  • The system is studied and linearized about the equilibrium points.
  • State space representation is derived from the linearized model.
  • Conditions under which the system is Controllable and Observable is studied.
  • LQR controller is designed and it's stability is studied using Lyapunov stability analysis.
  • Luenberger Observer is designed and studied for multiple output combinations available to observe.
  • Finally, and LQG controller is designed for the Linearized system and also tested on the original non linear model.

Mobile Robot design and control using ROS

In this project, a 4-wheel drive mobile robot is designed in SOLIDWORKS, and the robot is controlled using ROS Noetic and simulated in GAZEBO.

SKILLS Used: SOLIDWORKS, ROS Noetic, Python, OOP

  • A toy car model is designed in SOLIDWORKS, where all the wheels can be controlled independeptly.
  • The car model is then exported to URDF using URDF exporter plugin in SOLIDWORKS.
  • The URDF is modified to add a lidar and appropriate controllers for all the joints are also added.
  • A XACRO file created to integrate the LIDAR and controllers with the existing URDF to generate a new URDF file.
  • Launch file is created to spawn the robot in Gazebo in a modified world, and the model is tested with teleop operation.
  • Finally Publishers and Subscribers are written to follow a simple path in the arena.

April 2019

SAE Aerodesign West 2019

This project was done as a team by Team PHOENIX, the official Aeromodelling club of NIT Jamsedpur.

SKILLS Learnt: Team management, leadership, sponsorship, teaching, aeromodelling, airflow analysis, SOLIDWORKS, ANSYS, XFLR5, MATLAB 3D graphs, fabrication (wood and aluminium), testing sequence design, control system design, troubleshooting failures, circuit design, BLDC and servo motor circuits, Radio Control design and operation.

  • I was the Technical Head of Team PHOENIX for the year 2018-2019, and also for the SAE Aerodesign West 2019.
  • As Technical head, I led the complete calculations and design team, and the fabrication of the aircraft model.
  • We started with the Conceptual design based on Problem statement restrictions and requirements. I did the final conceptual model, and the form of wing design (backswept), tail design and fuselage mounting of wings.
  • Preliminary design and calculations was done to validate the design of the model. This was done iteratively until a good theoretical value of lift was achieved. I lead the preliminary calculations for lift and drag coefficients.
  • Then the detail design in CATIA was done by the design team members, and I did the material selection for each part of the plane, and started working on the electrical components specifications.
  • Electrical components was selected based on the lift requirements from the calculations, and ANSYS analysis of the model. I checked multiple motor and propellor combinations and selected the one with highest thrust with lowest current requirement.
  • Once the configuration was finalized for the complete aircraft, I started the fabrication with a team of 5 juniors under me.
  • Balsa wood is used for the complete body of the aircraft, except the fuselage which was made from Aluminium.
  • Once the fabrication is completed, we did a complete mounting of all the equipments, and used tennis balls to imitate humans in the aircraft, with steel plates as payload.
  • In the first flight test, the plane lifted the complete weight that we estimated from calculations, and flight round was successful.
  • At the final event in Van Nuys, we successfully completed 3 flight rounds, and we secured 9th rank globally amonst all the other teams.
  • We also secured 1st rank for our Design Report and approach to design the Aircraft, and the tools that we used to validate every result.

April 2018

Ojass 2018 at NIT Jamshedpur

Ojass is the annual technical fest at NIT Jamshedpur, and my team participated in multiple robotics competition and secured positions in all the events as described below.

SKILLS Learnt: Robot modeling, Arduino coding, sensor callibration, maze solving algorithms, controller design, wiring, fabrication

Bot-E-Maze: Autonomous Robot Competition

  • For this competition, an autonomous robot was supposed to follow a maze made up of black line and walls, and the robot should find the final goal.
  • We did the project using Arduino Uno, and used 5 IR sensors for detecting the lines and the walls that the robot should follow.
  • The inputs from all the IR sensors were used to design the Algorithm that can follow the line or wall. We used left wall follower algorithm for the wall follower.
  • The competition arena was a 2 floor arena, and our robot was able to complete the eitire competition, and we won 2nd position among the 75+ teams that participated.

Shape Shifter: A mobile robot with manipulator

  • For this competition, a mobile robot with a mounted arm should be used to complete the given tasks and reach a final goal.
  • We designed a four wheel drive robot, and designed a 4 DOF arm to be used to pickup and place objects.
  • The competition arena had balls, square boxes, inclined slopes and rotating floor as hurdles, and we were able to complete all the challanges.
  • We won the event with 1st position among the 40+ teams that participated in the event.

Mac Fifa: Robot playing Football

  • For this competition, a mobile robot should be used to play the game of football/soccer.
  • We used our robot from the above event, and modified it to have more weight and a lower Centre of Mass.
  • We used variable voltage regulator for this event to have more control on the robot speed and motion, and we made a custom controller with DPDT switches to control the robot for this event.
  • We won the event with 3rd position among the 90+ teams that participated in the event.
  • We also participated in the similar event at a different Institute (RVS academy) where we had to use two robots, and there we won the event with 1st position among the 30+ teams that participated.
